桌面设置为中文时单独设置shell语言为英文

在sh配置文件(zsh：.zshrc / bash：.bashrc)添加或修改以下命令
修改配置文件后重启shell生效或source 配置文件

user is not in the sudoers file.

有一次不知道是不是错用了usermod，导致重新登录后user不在root组中
目前添加用户到root中有三种：

  1. 修改/etc/sudoers文件
    添加user ALL=(ALL:ALL) ALL
    但此时软件更新管理等可能还是无法通过管理认证，需要重新设置sudo密码 sudo passwd

  2. adduser（推荐）
    sudo adduser user sudo

    adduser会自动创建用户目录和密码，useradd不会

  3. useradd

都需要先获取root权限才可以，如果碰巧原本就一个用户…无法进入
那就需要借用另一个linux引导系统 / WSL2挂载linux盘 /diskgenius专业版 先修改sudoers文件再添加了
